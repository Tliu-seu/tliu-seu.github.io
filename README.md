# Tao Liu Homepage

This repository contains the source code for [tliu-seu.github.io](https://tliu-seu.github.io), the personal academic homepage of Tao Liu.

The site is built with [al-folio](https://github.com/alshedivat/al-folio), a Jekyll theme for academic websites.

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000>.

## Main Content

- `_pages/about.md`: homepage biography and profile block
- `_posts/`: blog posts
- `_bibliography/papers.bib`: publications in BibTeX format
- `_data/cv.yml`: CV content
- `_data/repositories.yml`: GitHub profile/repository cards
- `assets/img/profile.png`: profile image
